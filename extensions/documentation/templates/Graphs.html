SVG


Generates a SVG file using Matplotlib on the server and returns the result as string that is then set as inner html of div
HTML


Generates a HTML file using plotly on the server and returns the name that is then loded into the div with jquery
PLOTLY


Generates a plotly graph on the server and returns the .fig data (much smaller then the rendered svg) that is then rendered in the browser using plotly.js
This is the most flexible method and should be used if possible.
It Does not work together with D3.js so only either one can be used
